# 🌊 MAR ABAIXO QUEST

> Um clone simplificado do clássico *Seaquest* (Atari 2600), desenvolvido inteiramente em **C puro** para rodar direto no seu terminal!

---

## 🎯 Sobre o Projeto

O **Mar abaixo Quest** traz a nostalgia dos jogos retrô de 8 bits para a linha de comando. Este projeto foi desenvolvido com foco no aprendizado de programação estruturada em C, manipulação de buffers de terminal e trabalho colaborativo em equipe utilizando Git e repositórios remotos.

---

## 🎮 Controles

| Tecla | Ação |
| :---: | :--- |
| <kbd>Ctrl</kbd> | **Iniciar** o jogo |
| <kbd>▲</kbd> <kbd>▼</kbd> <kbd>◄</kbd> <kbd>►</kbd> | **Mover** o submarino |
| <kbd>Espaço</kbd> | **Atirar** / Disparar torpedos |

---

## ✨ Destaques e Funcionalidades

* 🕹️ **Gráficos em ASCII Art:** Animações e renderização retro diretamente na tela do terminal.
* 🛠️ **Desenvolvimento Nativo:** Sem frameworks externos — feito apenas com C e bibliotecas padrão (`GCC`/`MinGW`).
* 💻 **Suporte:** Compatível com APIs de terminal do **Windows**.
* 🔊 **Efeitos Sonoros:** Sonoplastia integrada usando recursos nativos do sistema.
